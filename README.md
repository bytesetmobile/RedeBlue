# RedeBlue
O aplicativo RedeBlue é uma plataforma didática e comunitária para o mapeamento colaborativo da água. Feito para estudantes, professores e cidadãos, o app permite registrar e compartilhar a percepção da comunidade sobre a água da sua região. Juntos, criamos um mapa vivo que estimula a educação ambiental e o debate sobre esse recurso.
